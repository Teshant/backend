# FastAPI Async Backend Service

Production-style asynchronous backend service built with FastAPI, demonstrating real-world backend patterns including authentication, webhook handling, and automated testing.

---

## Overview

This project is a fully asynchronous FastAPI backend designed to reflect how production backend services are structured and implemented.  
It focuses on clarity, maintainability, and realistic architecture rather than demo-level examples.

---

## Key Features

- Asynchronous FastAPI application
- JWT-based authentication
- Secure password hashing
- Order management workflow
- Webhook-based payment event handling
- Async SQLAlchemy ORM integration
- Pytest-based async test suite
- Clean, modular project structure
- GitHub-ready and recruiter-friendly

---

## Tech Stack

- **API Framework:** FastAPI (classic syntax)
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy (Async)
- **Authentication:** JWT (OAuth2 Password Flow)
- **Password Hashing:** bcrypt (passlib)
- **Testing:** pytest, httpx
- **Server:** Uvicorn
